# todo-list-test > todo-api

`todo-api` for todo app depends on -
1. apollo-server
2. express
3. graphql
4. mongoose

App has following subdirectory -
```
├───config
├───models
├───mutations
├───queries
├───services
└───types
```

Run local dev server.
```
npm install
npm run local
```

App is docker enabled as well.
